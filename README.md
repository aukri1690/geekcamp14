## 概要
自己紹介カードをオンラインで作成・共有できるサービスです。

## 想定する利用場面
大学の新入生ガイダンスなど初対面同士の交流の場<br>

## 開発の背景
新しいコミュニティで自己紹介カードを書く機会は少なくありません。<br>
しかし、1度書いても読まれないまま役目を終わることも珍しくないです。<br>
そこで、交流者同士の関係構築をサポートする役目を担えるよう、このプロダクトの開発に至りました。
